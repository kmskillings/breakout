# breakout
An implementation of the classic videogame Breakout, targeting the 10DE-Lite FPGA board, written in VHDL.

This project was originally assigned as the final project for ECE-5730 Reconfigurable Computing, taught by Dr. Jonathan Phillips at Utah State University in Fall 2023.

After the semester was over, I discussed the project with Dr. Phillips. Our discussion, and my own subsequent research, inspired me to attempt the project again, using new strategies and techniques. This repo is my new attempt.

This project is written in VHDL, targeting the 10DE-Lite FPGA board from Terasic, which features an Altera MAX 10 FPGA. Modelsim is used for simulation. Quartus Prime is used as the synthesis tool.
